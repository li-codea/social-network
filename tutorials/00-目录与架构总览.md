# 社交网络好友推荐系统 — 后端代码详解目录

## 项目简介

这是一个基于 **FastAPI + SQLAlchemy + MySQL** 的社交网络好友推荐系统后端。核心技术包括：

- **CTE 递归查询**：发现二度/三度好友网络
- **INTERSECT 运算符**：计算共同好友
- **JSON_OVERLAPS**：基于兴趣标签匹配用户
- **加权评分算法**：综合多维度信号生成推荐

## 架构分层

```
backend/
├── app/
│   ├── __init__.py              # App 包初始化
│   ├── config.py                # 配置层：数据库连接参数
│   ├── database.py              # 数据层：SQLAlchemy 引擎与会话管理
│   ├── main.py                  # 应用入口：FastAPI 实例化、路由注册、CORS
│   ├── models/                  # ORM 模型层（SQLAlchemy 表定义）
│   │   ├── __init__.py
│   │   ├── user.py              # User 表
│   │   ├── friendship.py        # Friendship 无向好友关系表
│   │   └── message.py           # Message 私聊消息表
│   ├── schemas/                 # Pydantic 数据校验层（请求/响应模型）
│   │   ├── __init__.py
│   │   ├── common.py            # 通用模型：分页、错误、消息响应
│   │   ├── user.py              # 用户相关 Schema
│   │   ├── friendship.py        # 好友关系相关 Schema
│   │   └── message.py           # 消息相关 Schema
│   ├── services/                # 业务逻辑层（核心算法）
│   │   ├── __init__.py
│   │   ├── user_service.py      # 用户 CRUD
│   │   ├── friendship_service.py# 好友关系管理 + 共同好友计算
│   │   ├── chat_service.py      # 消息收发与会话管理
│   │   └── recommendation_service.py  # 好友推荐引擎
│   ├── routers/                 # API 路由层（HTTP 端点）
│   │   ├── __init__.py
│   │   ├── users.py             # 用户相关端点
│   │   ├── friendships.py       # 好友关系端点
│   │   └── chat.py              # 聊天消息端点
│   └── utils/                   # 工具函数
│       ├── __init__.py
│       └── pagination.py        # 分页 offset/limit 计算
└── test_api.py                  # 12 个端到端 API 测试用例
```

## 数据流示意

```
HTTP 请求 → Router（路由层）→ Service（业务逻辑层）→ Model（ORM 数据层）→ MySQL
                ↕                      ↕
           Schema（校验层）        Schema（序列化层）
```
